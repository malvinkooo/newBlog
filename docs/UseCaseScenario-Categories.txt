# Сценарии вариантов использование для Блога по работе с разделами

#### Добавить раздел
##### Предусловие:
 1. Пользователь вошел в систему, используя свои логин и пароль для авторизации.
##### Основной сценарий:
 1. Пользователь выбирает пункт "Добавить раздел";
 2. Система отображает форму для добавления раздела;
 3. Пользователь вводит необходимые данные;
 4. Пользователь выбирает "Создать раздел";
 5. Система принимает ввод;
 6. Система сохраняет данные в БД;
 7. Система отображает диалоговое окно об успешной операции;
 8. Система отображает страницу с ранее добавленным разделом;
 9. Система ожидает дальнейших действий пользователя.
##### Альтернативные сценарии
 6а.1.1 Система отображает диалоговое окно с текстом ошибки по причине:
 - пользователь не имеет прав для добавления разделов;
 - пользователь ввел не все необх. данные;
 - пользователь заполнил все или часть полей не валидными данными;
 - произошла непредвиденная ошибка;
 6а.1.2 Система ожидает дальнейших действий пользователя.
